元件

本頁盤點全站實際引用中的 UI 元件，逐一列出使用情況與樣式規格，做為前端開發與設計協作的共同依據。每個元件依「讀碼 → 整理現況 → 提案 → 拍板」四步驟逐一定案：標「★ 待盤點」的尚未深入檢視；已有預覽圖的為現況記錄，規格同樣以定案為準。

Button

元件呼叫 19 處 / 9 檔 · 另有手刻 ~56 顆待收編

6 種 variant × 6 種 size，不再新增 variant。基底規格：文字 14px / 500 ( Button token )、圓角一律 md ( 8px )、預設高度 h-9 ( 36px )。主色與危險色分別走 --primary ( brand-500 ) 與 --destructive ( red-500 )，focus ring 為 brand-300 半透明。

主要動作 次要未使用 外框 Ghost 刪除 連結未使用
variant: default · secondary ( 未使用 ) · outline · ghost · destructive · link ( 未使用 )
size: default(h-9) · sm(h-8) · lg(h-10) · icon(36) · icon-sm(32) · icon-lg(40)
已拍板規則 ( 2026.7.19 )：icon 按鈕一律 size="icon-sm"，頁碼類用 size="icon"，不得用 class 覆寫尺寸；手刻的 brand 外框按鈕樣式廢除，收斂為 outline 或 default；全站約 56 顆手刻按鈕隨頁面遷移分批收編呼叫 Button ( 手刻 teal-600 與 brand-500 同值，收編視覺幾乎零變化 )；Tab 切換、篩選 pill、選擇卡等 ~18 顆不屬 Button 職責，歸各自元件。secondary 與 link 為 shadcn 原生保留，標示未使用

Input

元件呼叫 12 處 / 7 檔 · 另有手刻 11 顆待收編

單行輸入。h-9 ( 36px )、rounded-md、文字 14px，焦點時顯示 3px brand-300 焦點環。

預設輸入框 焦點狀態

已拍板規則 ( 2026.7.19 )：元件不加 size prop ( vendor 檔不 fork )，小尺寸定慣例 class="h-8 text-xs"；搜尋框 icon 空間用 pl-9、密碼顯示鈕用 pr-10 屬合理佈局覆寫；手刻 11 顆 ( 標準款 6、表格迷你款 4 吸附小尺寸慣例、無框搜尋款 1 ) 隨頁面遷移收編
