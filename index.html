資料匯流平台設計系統

所有資料匯流平台相關網站的正式設計系統首頁，收錄了顏色、文字、間距等 Design Token 規範，以及全站元件盤點。以下先介紹目前用到的技術棧。

前端框架

Vue 3，全站採用 Composition API 開發。

vuejs.org →

UI 元件基礎

shadcn-vue，透過 CLI 把原始碼複製進專案，而不是安裝成套件，方便客製化與 token 化。

shadcn-vue.com →

樣式引擎

Tailwind v4，Design Token 透過 @theme 區塊對應成 utility class。

tailwindcss.com →

字體

英文與數字用 Public Sans，中文用 Noto Sans TC。

Public Sans → Noto Sans TC →

圖示

Phosphor Icons，全站介面圖示採用這套。

phosphoricons.com →
