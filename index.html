資料匯流平台設計系統

DCF 存取節點平台的正式設計系統首頁，收錄了顏色、文字、間距等 Design Token 規範，以及全站元件盤點。以下先介紹目前用到的技術棧，再往下可以直接進入設計規範或元件頁。

正式設計系統 Vue 3 + shadcn-vue Tailwind v4 字體 Public Sans + Noto Sans TC 更新 2026.07.12

技術棧

本站與 nics-dcf-access-demo 共用以下技術，每張卡片都連到官方文件。

Vue 3

前端框架，全站採用 Composition API 開發。

vuejs.org →

shadcn-vue

UI 元件基礎，透過 CLI 把原始碼複製進專案，而不是安裝成套件，方便客製化與 token 化。

shadcn-vue.com →

Tailwind v4

樣式引擎，Design Token 透過 @theme 區塊對應成 utility class。

tailwindcss.com →

字體

英文與數字用 Public Sans，中文用 Noto Sans TC。

Public Sans → Noto Sans TC →

開始使用

正式規範分成兩個頁面，依需求進入，也可以直接點左側的「設計規範」「元件」展開子項目。

Foundation

設計規範

顏色、文字、間距、圓角、陰影、頁面佈局，以及狀態色票、審計清單。
Components

元件

全站實際引用中的元件盤點，依用途分組，逐一列出樣式規格。
